# V28.10 vs V28.11 能量收集修复对比

## 问题

V28.10版本在`get_harvesting_rate()`中强制设置最小收集率：
```python
total_rate = max(total_rate, 0.000001)  # ❌ 错误
```

这导致**午夜0点收集了0.3J能量**，而理论值应该是0J。

## 修复

V28.11版本改为**仅在使用模拟数据时应用最小收集率**：
```python
if self.solar_loader is None:
    total_rate = max(total_rate, 0.000001)  # ✅ 正确
# 使用NASA数据时，保持total_rate不变（可能为0）
```
