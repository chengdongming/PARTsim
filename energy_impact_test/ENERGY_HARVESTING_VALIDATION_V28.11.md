## V28.11关键修复

### 问题描述
V28.10版本在`get_harvesting_rate()`函数中强制设置了最小收集率：
```python
total_rate = max(total_rate, 0.000001)  # 错误：强制最小收集率
```

这导致即使辐照度为0（如午夜），系统也会收集能量：
- 最小收集率: 0.000001 J/ms
- 仿真时长: 300000ms
- 错误收集: 0.000001 × 300000 = **0.3J** ❌

### 修复方案
V28.11版本修改为：**只有在使用模拟数据时才应用最小收集率**
```python
if self.solar_loader is None:
    # 只在不使用真实数据时应用最小收集率
    total_rate = max(total_rate, 0.000001)
# 使用真实NASA数据时，保持total_rate不变（可能为0）
```

### 修复效果
- **V28.10**: 0点收集0.3J（错误）
- **V28.11**: 0点收集0J（正确）✅

### 能量收集率计算公式

```
收集率(J/ms) = 辐照度(W/m²) × PV效率 × PV面积(m²) / 1000
```

### 8点验证
```

收集率 = 541.2 × 0.18 × 1.0 / 1000

       = 0.097416 J/ms
       = 4.870530 J/50ms ✅
```
