## 🔧 应用的三个Bug修复

### Bug #1: 能量扣除允许负数
**修复前**:
```cpp
if (_current_energy >= total_energy_to_deduct - EPSILON) {
    _current_energy -= total_energy_to_deduct;  // 可能变负数
}
```

**修复后**:
```cpp
if (_current_energy >= total_energy_to_deduct) {
    _current_energy -= total_energy_to_deduct;  // ✅ 不允许负数
} else {
    _energy_depleted = true;  // ❌ 标记能量耗尽
    // 中断所有任务
}
```

### Bug #2: 缺少能量耗尽标志
**修复**: 添加 `_energy_depleted` 标志，在多个关键位置检查：
- `performTickScheduling()`: 能量耗尽时跳过调度
- `addTask()`: 能量耗尽时拒绝新任务

### Bug #3: 能量检查在扣除之后
**修复**: 先扣除能量，检查是否足够，不足则设置 `_energy_depleted` 并中断所有任务
