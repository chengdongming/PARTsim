# 场景3: 100J能量（3核4任务）- 手动仿真

## 配置
- CPU: 3核
- 初始能量: 100.0 J (100000 mJ)
- 任务: task_1(5ms), task_2(8ms), task_3(10ms), task_4(12ms)
- 单位能耗: 0.558 mJ/ms per task

---

## TIE手动仿真（简化版）

### 0ms
```
能量: 100000 mJ
调度: task_1, task_2, task_3（前3个）
```

### 5ms
```
能量: 100000 - 5×1.674 = 99163 mJ
事件: task_1完成
调度: task_4（填补空闲CPU）
```

### 8ms
```
能量: 99163 - 3×1.116 = 98818 mJ
事件: task_2完成
调度: 无新任务（只有task_4在等待队列）
```

### 10ms
```
能量: 98818 - 2×0.558 = 98707 mJ
事件: task_3完成
调度: 无新任务
```

### 17ms
```
事件: task_4完成
调度: 无新任务
```

### 20ms
```
事件: task_1新实例到达（周期20ms）
调度: task_1
```

...后续周期任务继续执行

### 预期结果（运行20ms）
```
任务完成数: 3（task_1, task_2, task_3各完成1次实例）
Tick次数: 20
总能耗: 约11.5 mJ
剩余能量: 99988.5 mJ
```
