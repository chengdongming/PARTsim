## 🔧 应用的三个Bug修复

### Bug #1: 能量扣除允许负数
**修复位置**：
- [librtsim/scheduler/gpfp_tie_scheduler.cpp:871-920](../../../librtsim/scheduler/gpfp_tie_scheduler.cpp#L871-L920)
- [librtsim/scheduler/gpfp_btie_scheduler.cpp:1475-1506](../../../librtsim/scheduler/gpfp_btie_scheduler.cpp#L1475-L1506)
- [librtsim/scheduler/gpfp_tgf_scheduler.cpp:1393-1424](../../../librtsim/scheduler/gpfp_tgf_scheduler.cpp#L1393-L1424)

**修复内容**：
```cpp
// ❌ 修复前
if (_current_energy >= total_energy_to_deduct - EPSILON) {
    _current_energy -= total_energy_to_deduct;  // 允许负数
}

// ✅ 修复后
if (_current_energy >= total_energy_to_deduct) {
    _current_energy -= total_energy_to_deduct;
} else {
    _energy_depleted = true;  // 设置标志
    // 中断所有任务
}
```
